Photo Sharing App

Code

This project used JavaScript, HTML, and CSS to create a photo sharing app where users can engage with content through likes, comments, bookmarks, and follows. The user is recommended accounts to follow and has stories to view. When looking at a post, the user has the option to expand it and see it in more detail. When the user is done, they can sign out of their account.
